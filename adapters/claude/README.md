# Claude Code capability adapter

This package discovers an executable and runs only bounded `--version` and
`--help` probes. A configuration directory or lifecycle hook is not evidence of
an editable prompt surface. Stock-TUI inline rendering therefore remains
disabled unless a future documented native handshake proves every required
buffer, cursor, event, decoration, insertion, completion-awareness, and disposal
dimension.

Experimental adjacent PTY support can be reported only for a protocol-valid
profile whose executable path, version, and SHA-256 fingerprint match exactly.
Any changed or unknown binary fails closed. Probes never read settings,
credentials, session data, or prompt content and never start a model request.
