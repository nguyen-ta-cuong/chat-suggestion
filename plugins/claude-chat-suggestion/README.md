# Chat Suggestion for Claude Code

This is a Claude Code companion plugin, not a stock-TUI editor replacement.
Claude Code plugins can add skills, agents, hooks, and MCP servers, but this
plugin does not receive the live prompt buffer or cursor and cannot render or
accept Cursor-style ghost text while you type.

Load it with Claude Code's documented local plugin directory or a marketplace
entry. Use the `chat-suggest` skill after submitting a prompt when you want a
bounded continuation to review manually. Nothing is inserted or submitted
automatically, and no prompt or completion is logged by the plugin.
