# Code of conduct

We want this project to be a welcoming place for thoughtful technical work.

Be respectful, assume good intent, give actionable feedback, and focus critique
on ideas and code rather than people. Harassment, discrimination, threats,
doxxing, and sustained disruptive behavior are not accepted in project spaces.

Maintainers may edit or remove contributions that violate these expectations
and may temporarily or permanently restrict participation. Report conduct
concerns privately through the repository owner's GitHub profile. Maintainers
will handle reports with discretion and avoid unnecessary disclosure.
